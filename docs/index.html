📜

一番下まで丸ごと1枚

縦に長いページも、スクロールしないと見えない一番下まで丸ごと撮影。業務システムの長い検索結果や、縦長のLPも一発です。

🖼️

PNGとPDFで保存

画像（PNG）と、そのまま資料にできるPDFの両方に対応。2倍・3倍の高解像度撮影もできるので、ズームしても文字がくっきり。

🔑

ログインしたままのページもOK

専用ブラウザで普通にログインして使えます。ログイン状態は次回起動時も保持されるので、会員ページや管理画面の記録にも。

🖱️

操作はボタン1つ

撮りたいページで、画面右下のピンクの「PATTI SHOT」ボタンを押すだけ。設定も歯車マークから選ぶだけの2項目です。

🚀 はじめかた（3ステップ）

  1. 上の 「⬇ 無料ダウンロード」 を押す → PATTI_SHOT.exe が保存されます
  2. 保存した PATTI_SHOT.exe をダブルクリック → 専用のブラウザが開きます（インストール作業はありません）
  3. 撮りたいページを開いて、画面右下のピンクの「PATTI SHOT」ボタンをクリック → 撮影完了！保存フォルダが自動で開きます
⚠「WindowsによってPCが保護されました」と出たら：「詳細情報」→「実行」の順にクリックしてください。 これは署名のない個人開発アプリで必ず出る画面で、異常ではありません。

⚙ 設定（2つだけ）

ボタンの上の歯車マーク⚙を押すと設定が開きます。
保存形式（PNG / PDF / 両方）と画質（1x / 2x / 3x）を選ぶだけ。次回も覚えています。

🔃 アップデート

起動すると自動で新しいバージョンを確認します。
新しい版があると画面右下にお知らせが出るので、「更新する」を押すだけ。
ダウンロード → 置き換え → 再起動まで全自動です。

❓ よくある質問

撮った画像がどこかに送信されたりしない？

されません。撮影と保存はすべてあなたのPCの中で完結します。通信が発生するのは「起動時の更新チェック（GitHubに最新版を確認）」と「Webページを見るための普通のブラウザ通信」だけです。

動作環境は？

Windows 10 / 11 です。ブラウザは、PCにインストール済みの Google Chrome を使います（Chrome推奨）。Chromeが無い場合は、初回だけ専用ブラウザを自動で取得します（操作は不要です）。

「WindowsによってPCが保護されました」と出て起動できない

「詳細情報」→「実行」の順にクリックしてください。署名のない個人開発アプリで必ず出る表示で、異常ではありません。

ウイルス対策ソフトが警告を出すことがある？

ごくまれに誤検知されることがあります（このアプリの作り方＝PyInstaller製のexeで起こり得る既知の現象です）。ソースコードは全部GitHubで公開しているので、中身は誰でも確認できます。

ログインが必要なページも撮れる？

撮れます。開いた専用ブラウザで普通にログインすれば、見えている画面がそのまま撮影されます。ログイン状態は次回起動時も保持されます。

どのくらい長いページまで撮れる？

数万ピクセル級の非常に長いページまで対応しています。あまりに長いページでは、PDFは規格の上限に合わせて自動で複数ページに分かれます（文字の行が切れない位置で分割されます）。

撮影に失敗したら？

空白などの失敗を自動で検出して、条件を変えて自動でリトライします。それでも失敗した場合だけ、画面に日本語で理由が表示されます。

Macで使える？

現在はWindows専用（Windows 10 / 11）です。

商用利用できる？

できます。ただし、撮影したページの内容の権利は各サイトにあります。社内記録・資料などの範囲でご利用ください。

なぜ無料なの？怪しくない？

「クリエイターがちょっと助かるツールを無料で公開する」という STUDIO PATTI の活動の一環です（このページの広告枠などで運営しています）。全ソースコードをGitHubで公開しているので、中身は誰でも確認できます。
