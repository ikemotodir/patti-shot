# PATTI SHOT v2.0

STUDIO PATTI製のChrome拡張機能です。
いま見ているページを、**スクロールしないと見えない一番下まで丸ごと**撮影して、高画質のPNG画像とPDFで保存します。

- 操作は **2クリックだけ**(アイコンをクリック → 「撮影」を押す)
- 保存先はいつもの**ダウンロードフォルダ**
- ログインして見ているページも、**見たままそのまま**撮れます

---

## 使い方(ふだんはこれだけ)

1. 撮りたいページを開く
2. 画面右上のツールバーにある **PATTI SHOTのアイコン(ピンクのカメラ)** をクリックする
3. 大きな **「撮影」ボタン** を押す
4. 「ページ読み込み中…」→「撮影中…」→「保存完了！」と表示されたら終わり
5. ダウンロードフォルダに `PATTISHOT_サイト名_日付_時刻.png` と `.pdf` ができています

### 覚えておくと便利
- 「撮影」ボタンの下のチェックで、**PNGだけ / PDFだけ / 両方** を選べます(次回も覚えています)
- **すごく長いページ**は、画質を落とさないために PNGが `_1.png` `_2.png`… と数枚に分かれて保存されます(その時は画面にお知らせが出ます)

---

## 撮影中に出る表示について(こわくないやつ)

- 撮影中だけ、画面の上に **「PATTI SHOT がこのブラウザをデバッグしています」** という帯が出ます。
  → **撮影のための正常な表示です。** 撮影が終わると自動で消えます。
- Chromeを起動したとき **「デベロッパーモードの拡張機能を無効にしてください」** という警告が出ることがあります。
  → 開発者向けの読み込み方をしているための**仕様**です。「×」で閉じてそのまま使ってOKです。
- `chrome://〜` のページやChromeウェブストアなど、Chromeのルールで撮影できないページでは「このページは撮影できません」と表示されます。

## インストール手順(最初の1回だけ)

1. この `patti-shot` フォルダを、消してしまわない場所に置く(例: ドキュメントの中)
2. Chromeを開き、アドレスバーに `chrome://extensions` と入力してEnterを押す
3. 画面**右上**にある **「デベロッパーモード」** のスイッチを **ON** にする
4. 画面**左上**に現れた **「パッケージ化されていない拡張機能を読み込む」** ボタンを押す
5. さっき置いた `patti-shot` フォルダを選んで「フォルダーの選択」を押す
6. 一覧に **PATTI SHOT** が出たら成功!
7. (おすすめ)ツールバーのパズルピース🧩マークをクリックして、PATTI SHOTの**ピン📌**を押しておくと、いつでもワンクリックで使えます

---

## 更新のしかた(新しいバージョンが出たら)

1. `patti-shot` フォルダの中にある **`update.bat` をダブルクリック** する
2. 黒い画面が出て、自動で最新版に置き換わります(「更新完了!」と出るまで待つ)
3. Enterキーを押すと、Chromeの拡張機能ページ(`chrome://extensions`)が自動で開きます
4. **PATTI SHOT の「更新ボタン(🔄)」** を押す
5. ポップアップのバージョン番号が新しくなっていたら完了!

※ もし黒い画面に「エラー」と出たら、インターネット接続を確認して、もう一度ダブルクリックしてください。

---

## 困ったときは

| 症状 | 対処 |
|---|---|
| アイコンが見当たらない | ツールバーのパズルピース🧩 → PATTI SHOTのピン📌を押す |
| 「このページは撮影できません」と出る | `chrome://〜`やウェブストアなどChromeのルールで撮影不可のページです。普通のWebページで試してください |
| 撮影が終わらない | ページが非常に長いと数分かかることがあります。「撮影中…(12/48)」のように進み具合が出ていれば正常です |
| 保存したファイルが見つからない | Chromeのダウンロードフォルダ(`Ctrl+J`で履歴表示)を確認してください |
| それでもダメ | Chromeを再起動 → `chrome://extensions` → PATTI SHOTの🔄を押してから、もう一度撮影 |
